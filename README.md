# WeatherWag
### Where Weather and Dog Care Wag Together
WeatherWag is the ultimate app for dog owners who want simple, reliable weather information paired with useful tips to help their dogs enjoy every day outdoors.
## Tech Stack
- Flask (backend framework)
- OpenWeatherMap API (weather data)
- Google Maps API (location and parks)
- Gemini AI (personalized recommendations)
- HTML, CSS, JavaScript (frontend)

## Features
- Real time weather data using OpenWeatherMap API.
- Color coded message that recommends whether it's comfortable for a walk or if it's better to stay inside and play.
- Weather based product suggestions that keep your dog comfy.
- General Safety tips based off temperature.
- AI Integration using Gemini AI: provides personalized, weather based dog safety tips.
- Interactive map using Google Maps API: displays dog-friendly parks, dog parks, and trails filtered by type.
- Background color that changes based on weather.

## Why You'll Love WeatherWag
No more second guessing when it comes to your dog's outdoor time. Whether it's hot, pouring rain, or chilly and windy, WeatherWag gives you the info you need at a glance, helping you make smarter decisions to keep your dog happy. Plus with Gemini AI, you get personalized recommendations tailored to your dog's breed and needs.
